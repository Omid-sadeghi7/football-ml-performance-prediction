**Overview**

This project presents a machine learning solution for forecasting football players’ match involvement and physical workload using real-world performance and physical data. The objective is to support coaching staff in decision-making related to squad rotation, workload management, and match preparation.

**Data**

The project integrates two datasets, including physical performance data (e.g., sprint distance, speed, intensity zones) and match performance statistics (e.g., minutes played, goals, assists). These datasets were merged and engineered to create meaningful predictors for modelling.

**Methodology**

A full machine learning pipeline was developed, including data cleaning, feature engineering, and model training. Multiple models were implemented and compared, including Random Forest, XGBoost, and Gradient Boosting.

Hyperparameter tuning was performed using GridSearchCV, and models were evaluated using MAE, RMSE, and R². Additional scenario-based analysis was conducted to simulate real-world situations such as player rotation, fixture congestion, and injury recovery.

**Results**

The models demonstrated strong predictive performance in estimating players’ minutes played and workload distribution. Scenario analysis showed the model’s ability to adapt to different match conditions and provide actionable insights for decision-making.


**Skills Demonstrated**

Machine Learning, Feature Engineering, Model Evaluation, Scenario Simulation, Sports Analytics
